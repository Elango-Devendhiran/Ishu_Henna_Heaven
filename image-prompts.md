## Usage Notes

- Keep every image within the same warm gold-on-black lighting family so the site feels cohesive.
- For real client work, always get written model/photo consent before publishing bridal photography.
- If generating with tools that support aspect ratio, use **4:5** for gallery/portrait shots, **16:9** for hero/backgrounds, and **1:1** for Instagram-strip thumbnails.
